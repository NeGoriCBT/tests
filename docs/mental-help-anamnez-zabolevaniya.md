# Анамнез заболевания (Mental Help)

В разделе Word документа анкеты он называется **«Анамнез заболевания»**. В приложении возможны **два формата**: свободные текстовые вопросы (основная анкета v1 и v2) и **структурированный** блок (отдельная страница «Анамнез заболевания»). Они **не комбинируются** на одной странице: на `mental-help-v2.html` в этом разделе попадают только ответы **В1–В7**; структурированный JSON используется на `mental-help-disease.html`.

Исходники: `mental-help-data.js` (шаги В1–В7), `mental-help-v2-disease.js`, `mental-help-disease-drugs.js`, `mental-help-disease-data.js`, сборка Word — `buildWordBlockBody` в `mental-help-data.js`.

---

## 1. Свободный текст (шаги `disease-v1` … `disease-v7`)

Используется на страницах с полным мастером (`mental-help.js`, `mental-help-v2.js`). Каждый шаг — одно многострочное поле; **в Word** непустой текст шага добавляется в раздел «Анамнез заболевания» **как есть** (без шаблонных фраз), несколько шагов разделяются пустой строкой.

| ID шага | Метка | Вопрос (prompt) |
|---------|-------|-----------------|
| `disease-v1` | В1. | Когда вы впервые заметили, что что-то идёт не так? Опишите самые первые симптомы или ощущения. |
| `disease-v2` | В2. | Было ли что-то, что, по вашему мнению, могло запустить это состояние? Отметьте всё, что кажется важным. Если таких причин несколько — напишите их через запятую. |
| `disease-v3` | В3. | Пробовали ли вы справляться сами? Что именно делали? |
| `disease-v4` | В4. | Обращались ли вы раньше к специалистам в связи с этим состоянием и когда? Если их было несколько — напишите через запятую. |
| `disease-v5` | В5. | Принимали ли вы когда-либо препараты по назначению врача для улучшения психического состояния? Если их было несколько — напишите через запятую. |
| `disease-v6` | В6. | Как менялось ваше состояние от первых признаков до настоящего момента? |
| `disease-v7` | В7. | Опишите ваше состояние на сегодняшний день. Что беспокоит прямо сейчас сильнее всего? |

У каждого шага в интерфейсе есть **пример ответа** (`example` в `MH_STEPS`).

## 2. Структурированный анамнез (`disease-structured`)

Страница: `mental-help-disease.html` (шаги `MH_STEPS_DISEASE_ONLY` в `mental-help-disease-data.js`).

### 2.1. Хранение

- **Ключ в объекте ответов:** `disease-structured`.
- **Формат:** одна JSON-строка. Поля по умолчанию задаёт `emptyDiseaseStructuredState()` в `mental-help-v2-disease.js`.

| Поле | Тип | Назначение |
|------|-----|------------|
| `debutMode` | строка | `monthYear` \| `age` \| `unknown` \| `""` |
| `debutMonth` | строка 1–12 | месяц дебюта |
| `debutYear` | строка | год |
| `debutAge` | строка | возраст дебюта (полных лет) |
| `onset` | строка | `gradual` \| `acute` \| `unknown` |
| `earlySymptoms` | строка | выбранные подписи чекбоксов (вопрос 3), через перевод строки `\n` |
| `stressors` | массив строк | коды из таблицы стрессоров |
| `stressorsOther` | строка | уточнение для пункта «Другое» |
| `priorDoctor` | строка | `no` \| `yes` |
| `priorVisits` | массив объектов | визиты, если `priorDoctor === "yes"` |
| `priorMeds` | строка | `no` \| `yes` |
| `medEntries` | объект | ключ = id препарата из каталога, значение — поля ниже |

Объект визита `priorVisits[]`:

| Поле | Описание |
|------|----------|
| `specialist` | код из списка специалистов |
| `customOther` | текст, если выбран «Другой» |
| `reason` | по какой причине обращались |
| `reasonUnknown` | `true` — не помнит причину |

Объект `medEntries[id]`:

| Поле | Описание |
|------|----------|
| `months` | срок приёма в месяцах (число в строке) |
| `maxDose` | макс. суточная доза (текст) |
| `effect` | `""` \| `yes` \| `partial` \| `no` |
| `sides` | описание побочных эффектов; пусто → в Word подставляется «не отмечал(а)» по полу |
| `trade` | торговое наименование |

### 2.2. Вопросы на экране (6 блоков)

**Вопрос 1.** Когда Вы впервые заметили симптомы заболевания?

| Вариант | `debutMode` | Дополнительно |
|---------|-------------|----------------|
| Месяц и год | `monthYear` | выбор месяца (название по-русски), года (1950…текущий) |
| Возраст (лет) | `age` | число 0–120 |
| Не помню | `unknown` | — |

**Вопрос 2.** Как началось заболевание?

| Подпись | `onset` |
|---------|---------|
| Постепенно | `gradual` |
| Быстро | `acute` |
| Не помню | `unknown` |

**Вопрос 3.** Что Вас беспокоило в начале заболевания? — кнопка «Выбрать» → чекбоксы; выбранные пункты (только текст до тире) сохраняются в `earlySymptoms` списком, разделённым переводами строк (`\n`).

**Вопрос 4.** Что происходило в жизни незадолго до начала или перед ухудшением? — чекбоксы (мультивыбор). Если отмечено **«Ничего из перечисленного»** (`none`), остальные снимаются. Для **«Другое»** — поле уточнения `stressorsOther`.

| Код | Подпись в форме |
|-----|-----------------|
| stress | Стресс (без уточнения) |
| burnout | Психологическая перегрузка / выгорание |
| breakup | Расставание с партнёром / развод |
| death | Смерть близкого человека |
| jobloss | Потеря работы / увольнение |
| conflict | Конфликт в семье или на работе |
| somatic_illness | Физическая болезнь (+ поле «название болезни») |
| somatic_trauma | Травма (+ поле «характер травмы») |
| pregnancy | Беременность / роды |
| finance | Финансовые трудности |
| move | Смена места жительства / переезд |
| legal | Юридические проблемы / суд / тюрьма |
| none | Ничего из перечисленного |
| other | Другое |

**Вопрос 5.** Обращались ли к врачу по поводу симптомов раньше?

- **Нет** → `priorDoctor: "no"`.
- **Да** → `priorDoctor: "yes"`, список визитов (кнопки «Добавить визит», «Удалить»). У каждого визита:

| Специалист (`specialist`) | Подпись в форме |
|---------------------------|-----------------|
| psychiatrist | Врач психиатр |
| psychotherapist | Врач психотерапевт |
| neurologist | Врач невролог |
| psychologist | Психолог |
| therapist | Терапевт / семейный врач |
| other | Другой (+ текст в `customOther`) |

Дополнительно: чекбокс «Не помню причину», поле «По какой причине обращались?».

**Вопрос 6.** Назначалось ли лечение по поводу симптомов ранее?

- **Нет** → `priorMeds: "no"`.
- **Да** → для каждого препарата из каталога можно отметить приём и заполнить: срок (мес.), макс. доза, торговое наименование, выпадающий список «Стало лучше?» (да / частично / нет / пусто), поле побочных реакций.

Каталог препаратов — ниже; идентификаторы совпадают с ключами в `medEntries`.

## 3. Как JSON превращается в текст Word

Функция **`formatDiseaseStructuredForWord(state, gender)`** (`mental-help-v2-disease.js`) собирает абзацы; между ними вставляется **двойной перевод строки**. Учитывается **пол пациента** там, где нужны формы «обращался/обращалась», «не отмечал/не отмечала» в побочках.

**Первое предложение** (дебют + темп + стрессоры) — одна строка через запятую:

| Фрагмент | Условие | Пример |
|----------|---------|--------|
| Дебют | месяц + год | `Заболевание дебютировало в январе 2025 года` |
| Дебют | только год | `Заболевание дебютировало в 2025 году` |
| Дебют | только месяц | `Заболевание дебютировало в январе (год пациентом не указан)` |
| Дебют | возраст | `Начало заболевания с 8 лет` |
| Дебют | не помнит | `Не помнит дату начала заболевания` |
| Темп | постепенно / быстро / не помнит | `постепенно` / `быстро` / `темп нарастания симптомов не помнит` |
| Связь с событиями | выбрано «Ничего из перечисленного» | `ни с чем конкретным не связывает` |
| Связь | иначе | `на фоне <перечень в родительном падеже через запятую и «и»>` (например: стресса, потери работы; болезни/травмы с при необходимости `(со слов: …)`; «Другое» — `иного обстоятельства (со слов: …)`) |

Пример целиком: `Заболевание дебютировало в январе 2025 года, постепенно, на фоне стресса, потери работы и переезда.`

Далее — отдельные абзацы (через пустую строку в Word):

| Блок | Типичная фраза |
|------|----------------|
| Ранние симптомы (вопрос 3) | `В начале заболевания беспокоило: <категории>`. Текст собирает **`formatEarlySymptomsForWord`** (`mental-help-disease-early-symptoms-data.js`): блоки вида `по настроению — …; телесные симптомы — …` (см. `EARLY_SYMPTOMS_WORD_BLOCKS`), формулировки в **прошедшем времени** по карте `EARLY_SYMPTOM_PAST_BY_LABEL` (`mental-help-disease-early-symptoms-past.js`); после двоеточия строка приводится к **строчным** буквам (как в клиническом перечне). Резерв кода плоского перечня через `listWithAnd`: `src/backup/mental-help-disease-early-symptoms-word-v1.js`. |
| Не обращался к врачам | `По поводу данного состояния ранее к специалистам не обращался(лась).` |
| Обращался | `По поводу данного состояния ранее обращался(лась) к … и к …` (причина: по поводу «…» или причину не помнит). |
| Лечения не было | `Психотропная терапия по поводу данного состояния ранее не назначалась.` |
| Препараты заполнены | `Ранее отмечался приём психотропных препаратов:` далее пункты через точку с запятой. |
| Лечение без выбора препаратов | `Психотропное лечение назначалось; сведения о конкретных препаратах в анкете не указаны.` |
